# aio-cli-plugin-wxp
Create, Build, and Deploy WXP Applications.

# Usage
```sh-session
$ aio plugins:install @adobe/aio-cli-plugin-wxp
$ # OR
$ aio discover -i
$ aio wxp --help
```
